# Playfab

플레이 팹이란 게임 서비스에 필요한 플레이어 및 데이터 관리, 실시간 분석 등의 서비스를 제공하는 백 엔드(Back-end) 솔루션이다.

[MS PlayFab](https://docs.microsoft.com/ko-kr/gaming/playfab/what-is-playfab)

Playfab을 사용하기 위해서는 따로 [Playfab 계정](https://developer.playfab.com/en-US/my-games)을 등록해야 사용할 수 있다.

# Prerequisites

* Playfab 계정 등록, [Playfab 계정](https://developer.playfab.com/en-US/my-games)
* Unity3d 설치, [Unity3d](https://unity3d.com/kr/get-unity/download
* Playfab SDK 설치
